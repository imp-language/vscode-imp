# vscode-imp

[Imp](https://github.com/MH15/imp) support for Visual Studio Code.

# Features

- [x] Syntax highlighting
- [ ] Code completion
- [ ] Go-to-definition
- [ ] Linting
- [ ] Automatic formatting





# Development

```bash
npm install -g vsce
npm run yaml

vsce login imp-language # will prompt you to submit your Azure Personal Access Token
vsce package
vsce publish

```
